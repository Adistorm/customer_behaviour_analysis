📊 Customer Behaviour Analysis
📌 Overview

The Customer Behaviour Analysis project focuses on understanding purchasing patterns, customer preferences, and key business insights using data analytics techniques.
The project combines Python, SQL, and Power BI to clean, analyze, visualize, and present customer data in a meaningful way for decision-making.

This project demonstrates the complete analytics workflow — from raw data to actionable insights.

📁 Dataset

The dataset contains customer shopping behavior information, including:

Customer demographics

Product categories & items purchased

Purchase amount & frequency

Ratings & reviews

Payment & shipping methods

Discounts & promo usage

The dataset is used to identify trends, high-value customers, and purchasing patterns.

🛠 Tools & Technologies

Python → Data loading, cleaning, and EDA

Pandas, NumPy, Matplotlib, Seaborn → Data analysis & visualization

PostgreSQL → Querying & business insights

Power BI → Interactive dashboard & data visualization

Gamma → Presentation creation

Jupyter Notebook → Analysis workflow

⚙️ Project Steps
1️⃣ Data Loading

Imported dataset using Pandas

Inspected structure and data types

2️⃣ Data Cleaning

Handled missing values (e.g., review ratings)

Removed duplicates

Standardized column names & formats

3️⃣ Exploratory Data Analysis (EDA)

Analyzed customer demographics

Identified top-selling categories

Examined seasonal purchase patterns

Studied rating distributions and spending trends

4️⃣ SQL Analysis (PostgreSQL)

Key business queries performed:

Top selling product categories

Most frequently purchased items

High-value customers

Discount impact on purchase behavior

Seasonal buying trends

5️⃣ Power BI Dashboard

Created an interactive dashboard to visualize:

Sales by category

Customer segmentation

Purchase trends over time

Top products & revenue contributors

Discount vs purchase behavior

📊 Dashboard Highlights

The Power BI dashboard enables stakeholders to:
